# Password Generator

#python #password-generator #security #cli #cs50 #beginner-project

A simple command-line Python tool that generates strong random passwords based on user preferences.

## Features
- Choose length (6–32 characters)
- Optional numbers and special symbols
- Guarantees at least one number/symbol if selected
- Shuffles characters for natural-looking passwords

## How to Run
- git clone https://github.com/ccmgbajah-maker/password_generator.git
- cd password_generator
- python password_generator.py

## What I Learned
- Input validation try/except and loops
- String concatenation for character pools
- Using random.sample(), random.choice(), random.shuffle()
